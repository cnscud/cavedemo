# 脚手架

## 涵盖功能:
* > 使用Zookeeper作为注册中心
* > 数据库操作MyBatis

* 配置中心: Zookeeper

* 缓存操作 Redis
* 队列操作 Kafka: Producer, Consumer
* MyBatis配置存到Zookeeper
* 
* 登录授权
* 模版: Freemarker支持


## 相关工具
    代码生成: Mybatis Generate 或者使用 Free Mybatis Plugin (Idea)
